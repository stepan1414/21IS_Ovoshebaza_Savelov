## 📖 1. Описание проекта

Данный проект представляет собой информационную систему «Овощебаза»,
предназначенную для автоматизации учёта продукции, поставок и складского
хранения.

Система позволяет: - хранить данные о продукции и её сортах; - учитывать
поставки и остатки на складе; - анализировать движение товаров; -
получать отчёты через SQL-запросы; - работать через Python-приложение с
интерфейсом.

------------------------------------------------------------------------

##  2. Техническое задание (ТЗ) вкратце
*   **Цель:** Цифровизация учета овощей: от ручных журналов к реляционной базе данных на SQLite.
*   **Основные функции:**
   1. Регистрация поставщиков и поступлений товара.
   2. Учет проданных единиц продукции.
   3. Контроль текущих остатков на складе.
   4. Формирование отчетов по выручке за период.
*   **Требования к БД:** Минимум 4 связанные таблицы, соблюдение целостности данных.

------------------------------------------------------------------------

## ⚙️ Основной функционал

-   📦 Управление складами (добавление, редактирование, удаление)
-   🥕 Управление товарами (продукция и сорта)
-   🚚 Учёт закупок и поставок
-   📊 Контроль остатков на складе
-   📈 Формирование простых аналитических отчётов
-   🔍 Поиск и фильтрация данных

------------------------------------------------------------------------

## 🗄️ 3. Структура базы данных (ER-модель)

-   Продукция --- основные овощи (название, базовая цена)
-   Сорта --- разновидности продукции
-   Склады --- места хранения
-   Закупки --- операции поступления товара

ER-диаграмма находится в папке /diagrams

------------------------------------------------------------------------

## 🧠 4. Реализованные SQL-запросы

-   Подсчёт остатков товара
-   JOIN-запросы
-   Фильтрация по поставщикам и датам
-   Топ-3 самых продаваемых овощей
-   Отчёты за период
